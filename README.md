## 只求极致

[ **M** ] arkdown + E [ **ditor** ] = **Mditor**   

## 简单说明

Mditor 目前有两个版本
- 独立的桌面版本，目前仅有 Mac 版本。 
- 可嵌入到任意 Web 应用的 Embed 版本，通常用于「技术类博客、论坛、留言板」等应用中
